Side Tabs

Note that the JS and CSS are not minified so they are readable. See side-tabs.js for the tab creation code.

This example demonstrates side-aligned tabs in Ext JS. Tab Panels can be configured with a tabPosition of "right" to rotate the tab bar clockwise, or "left" to rotate the tab bar counter-clockwise.
