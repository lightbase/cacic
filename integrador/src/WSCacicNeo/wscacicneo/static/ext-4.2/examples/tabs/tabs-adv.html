Advanced Tabs

Note that the JS and CSS are not minified so they are readable. See tabs-adv.js for the tab creation code.

This TabPanel is built entirely with JavaScript and demonstrates:

  • Auto tab resizing
  • Tab scrolling
  • Tabs with icons
  • Tab plugins (context menu)
  • Adding tabs with JS
  • Changing a tab's closable property dynamically (via the context menu)
  • Disabling a tab (also via the context menu)
