CACIC es capaz de crear un diagnóstico preciso del parque computacional y disponibilizar informaciones como el número de equipamientos en su distribución en los mas diversos órganos, los tipos de softwares utilizados y licenciados, configuraciones de hardware, entre otras. También puede crear informaciones patrimoniales y de localización física de los equipamientos, ampliando el control del parque computacional y la seguridad en la red.

Este instalador tiene como objetivo facilitar el proceso de instalación y configuración de CACIC. Los detalles son presentados a continuación.

Pasos para la instalación:

  1. Introducción - Esta página!
  2. Licencia - Lea y acepte la licencia de CACIC.
  3. Pré-requisitos - Verificación de los requisitos para el CACIC-Gerente.
  4. Configuraciones - Datos para configurar el servidor de base de datos.
  5. Administración de CACIC-Gerente - Agregar el local y usuário administrador.
  6. Serviço FTP - Verificação básica de conexão ao serviço FTP.
  7. Finalización de instalación - Verificación de la instalación e criação do arquivo de configuração.

Recursos del Instalador:

  • Verifica versiones mínimas de softwares/bibliotecas para la ejecución de CACIC.
  • Creación de la base de datos.
  • Creación de las tablas.
  • Verifica y crea las configuraciones básicas para la ejecución de CACIC.
  • Incluir los datos para la demonstración del producto (opcional).
