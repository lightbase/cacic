CACIC is a computer inventory system. It is a group of software able to allow both hardware and software inventory. Initialy developed by Dataprev-ES, but nowdays it has lot of community contributions.

This web installer main goal is provide you an easy way for both install and setup CACIC base. See details bellow.

CACIC-2.6 (Kurumí kwoiti) - key news:

  • Now the Windows Agent as service.
  • A custom VNC remote support.
  • Windows 2008 and Seven® support introduced.

Installation steps:

  1. Introduction - This page!
  2. License - CACIC License, read and accept is needed.
  3. Prerequisite - Verify some CACIC-Manager (server) requisites.
  4. Setup - Database sever and basic data configuration.
  5. FTP Service - Verify FTP service connection.
  6. CACIC-Manager administrator - Add both administrator user and main office (headquarter).
  7. Finish installation - Check the installation process and build setup file.

Installer features:

  • Verify minimal both softwares and libraries to use CACIC-Manager.
  • Buil CACIC-Manager database.
  • Build database tables.
  • Setup check and build base data for CACIC.
  • Add database data for CACIC demo (optional).
