###########
# ATENÇÃO #
###########
 
A partir da versão JAN2006, o módulo gerente WEB do CACIC passou a usar criptografia AES(Advanced Encryption Standard), método RIJNDAEL-128bits, em suas trocas de informações com os agentes. Tornou-se necessário então a instalação do pacote "mcrypt".  Para distribuições Conectiva, recomendamos o endereço http://distro.conectiva.com.br/atualizacoes/?id=a&anuncio=000567 para obtenção do referido pacote.

Alertamos que sejam feitas prévias de utilização/atualização das novas versões em pequenas redes, para que seja evitada a eventual necessidade de retrabalho técnico, quanto à reinstalação manual dos agentes nas estações.

Os pacotes de instalação de agentes para o Sistema Operacional GNU/Linux poderão ser gerados a partir da descompactação e leitura do arquivo README do arquivo pyCACIC_Packages_Generator_2.6.0.2.zip existente na pasta raiz do módulo gerente web.
