Alguém poderia esclarecer quais as portas devem estar abertas nos clientes ( entrada e saída ) para que os dados sejam coletados ?

Os agentes (clientes) apenas enviam os dados coletados para o gerente (servidor) na porta TCP 80. Sendo assim, nos agentes não é necessário porta aberta para o recebimento de conexões.

Clientes não enviam informações

A variável register_globals no arquivo de configuração do php (php.ini)
deverá estar configurada como On
> register_globals = On
Se não estiver com essa configuração diversos erros irão ocorrer no
módulo gerente, e ele não irá conseguir escrever as informações no banco
de dados.

Além disso é necessário checar se o usuário do banco de dados que está
configurado no arquivo de configuração do gerente
(cacic2/include/config.php) possui acesso de escrita no banco de dados
do cacic.

segue um exemplo:
usuario="DOMINIO\administrador"
senha="senha do admin"
comando="c:\cacic2\cacic2.exe"
set Shell = CreateObject("WScript.Shell")
set Env = Shell.Environment("Process")
WinPath = Env("SystemRoot")&"\System32\runas.exe"
Shell.Run("runas /user:" & usuario & " " & CHR(34) & comando &

Alteração do Manual de Implantação
Como é do conhecimento de todos, a atualização dos agentes nos servidores dos ESXX, é realizada através do serviço ftp, daí a necessidade da informação do IP, usuário(repositorio) e senha para acesso de gravação no diretório /var/www/default/cacic2/repositorio/ do servidor do CACIC.

Após algum tempo percebemos a necessidade de algumas alterações no Manual de Instalação, até mesmo para preservar a segurança do próprio servidor. Por esse motivo, pedimos a gentileza de verificar e executar, se for o caso, as alterações abaixo, bem como anexar as mesmas na página 13, item "5.2 O SERVIDOR DE FTP DO MÓDULO GERENTE", do referido manual, para futuras consultas.

1) Criação do usuário "repositorio":
useradd repositorio -c "Repositório Cacic" -g www
Assim mantemos o padrão do servidor e apenas colocamos o usuário "repositorio" no grupo "www" para que ele tenha poderes total no /var/www/default/cacic2/repositorio/.

2) Criação de senha para o usuário "repositorio":
passwd repositorio

3) Alteração do dono, grupo e permissões, do diretório repositório do servidor cacic:
chown -R www:www /var/www/default/cacic2/repositorio/
chmod -R 2770 /var/www/default/cacic2/repositorio/
Obs: o ideal é que o www seja dono e grupo do /var/www/default/cacic2/ e todos arquivos abaixo dele. Questão apenas de organização.


4) Inserir o bloco abaixo no /etc/proftpd.conf para permitir a regravação, via ftp, dos arquivos existentes:

<Global>

<Directory /var/www/default/cacic2/repositorio>
AllowOverwrite on
</Directory>

<Directory /home/cacic/updates>
AllowOverwrite on
</Directory>

</Global>

A linha <Directory /home/*> é imprescindível nos servidores de updates pois a atualização dos servidores também é feita via ftp através do usuário cacic e sem essa linha não será permitida a regravação de arquivos pelo serviço ftp.

Casos do servidor de updates estar em uma máquina diferente do gerente.

Chamamos a atenção para os casos em que o servidor de updates está em uma máquina diferente do gerente. Nestes casos, deve ser observado se /etc/proftpd.conf contém a configuração abaixo:

<Global>

<Directory /home/cacic/updates>
AllowOverwrite on
</Directory>

</Global


No Cacic tem como saber o numero de estações com plenus(estação) duplicados?

No caso do Plenus temos duas coletas: uma para cópia plenus(ESXXXX) e outra para estação(ESXXXX/I). Acessar Estatísticas, selecione o "Estação Plenus" e o "Plenus for Windows".

informações. Algumas máquinas rodam e enviam mas vai somente as informações
basicas.

Verifique se o usuario cacic tem permissão de leitura e gravação no seu /home/updates.

Se você já fez o cadastro da rede no sistema, pelo menu MANUTENÇÃO => CADASTRO => REDES. O usuário (campo: "Usuário do Servidor de Updates) que você cadastrou deverá ter acesso (leitura, gravação e atualização) ao diretório cadastrado no campo "Path no Servidor de Updates", no servidor cadastrado no campo "Servidor de Updates(FTP)".

Procedimentos para alterar uma CAT errada

Quando tem uma informação errada na CAT, devemos alterá-la e não fazer seu cancelamento.
Pois após o cancelamento não se poderá cadastrar outra CAT com o mesmo NIT, CNPJ, DATA ACIDENTE e DATA ULTIMO DIA TRABALHADO (Se houver).
Uma CAT só pode ser alterada antes de ser vinculada a um benefício. Por isso, antes de vincular uma CAT a um benefício deve-se verificar se todas as informações estão corretas.

Para se alterar a DATA DO ACIDENTE somente através do aplicativo CATMAN, CAT na Intranet.
Seguem as informações para instalação do aplicativo:


É importante notar que todas APS que tem o SABI instalado, devem utilizar a CAT na Internet para cadastramento, e na Intranet para Alteração/ Cancelamento/ Reabertura. E o módulo no PRISMA fica desabilitado mesmo.
Mas as APS que não têm SABI podem instalar esses aplicativos também.
A instalação do aplicativo deverá ser feito, ou conduzido por um técnico da Regional da Dataprev.

Como proceder:
1) Entrar no www-disin, opção "Produtos DISI.N";

2) Clicar no módulo "CAT Intranet", e fazer a instalação (auto-explicativa).
Caso tenham dúvidas, os procedimentos foram juntamente com a versão 8.6;

3) Após a instalação da CAT - Intranet, o funcionário da APS, deverá verificar se a CAT está cancelada:

4) Em caso afirmativo, deverá primeiramente "REATIVAR" a CAT pela Intranet. Caso contrário poderá alterar a informação necessária para acerto da CAT;

5) Fazer a transmissão da CAT para o SUB, e aguardar seu retorno e posterior tratamento;

6) Só então vincular a CAT ao benefício.

Procedimentos para instalação do Netmeeting

1. Baixe o programa NetMeeting no seguinte endereço <<ftp://10.71.0.214/internet/nm30.exe>> ;
2. Execute o programa e confirme as mensagens de instalação até que apareça a mensagem que o NetMeeting foi instalado com êxito;
3. Após a conclusão da instalação execute o aplicativo que estará localizado no botão “Iniciar”, “Programas”, “NetMeeting” em seguida clique em avançar;
4. Quando forem solicitadas as informações pessoais, sugiro o seguinte:
Nome: o nome da APS
Sobrenome: o nº IP da máquina
Endereço eletrônico: o e-mail da APS
Local: o município/ES (exemplo: Vitória/ES)
Comentários: pode ficar em branco.
5. No próximo passo, desmarque as opções “Efetuar logon em um servidor ...” e “Não listar meu...” clique em “Avançar”;
6. Nesta tela iremos configurar a velocidade de conexão. Escolha “Rede local”;
7. Deixe marcadas as opções de “Colocar atalho na área de trabalho” e “Colocar atalho do NetMeeting na barra...”;
8. A próxima tela fará um alerta quanto ausência da placa de som, não se preocupe, clique em “concluir”. Se tudo correu bem, dentro de instantes o NetMeeting será iniciado;
9. Clique em “Ferramentas”, “Opções” e marque o item “Exibir o ícone do NetMeeting na barra de tarefas” . Na aba “Segurança” marque a opção “Prefiro fazer chamadas seguras”;
10. Para que o suporte remoto fique ativo, você deve ir em “Ferramentas”, “Compartilhamento Remoto de área de Trabalho”. Clique em “Avançar”, na tela de senha, coloque dataprev . Na hora de colocar a senha preste atenção, pois a senha deverá está em letras minúsculas;
11. Na tela seguinte o aplicativo solicita que seja ativada a proteção de tela. Marque a opção “Não, vou fazer isso mais tarde” e clique em “Avançar” logo em seguida conclua;
12. O último passo é clicar em “Chamada”, “Sair e ativar compartilhamento remoto da área de trabalho”.

Como detectar o momento correto para troca do cilindro OPC(Okipage)

A impressora informa através de seu painel o momento em que a vida útil do cilindro OPC se expirou, porém nem sempre existe a necessidade da troca deste cilindro. Para confirmar se existe realmente a necessidade da troca do cilindro, deve ser verificado os seguintes itens no cilindro:

· Quando o cilindro está ruim, a impressão fica toda manchada de preto, começando pelas bordas (como se fosse uma cópia de xerox), retire o cilindro da impressora e note que a parte verde (cilindro orgânico) está toda coberta de pó de toner, isto não é normal.
· Existe também o fato de começar a vazar toner pelas laterais do cilindro, pelo desgaste natural do mesmo e pelo excesso de toner que é depositado internamente em sua auto limpeza.
· Alguns defeitos podem ocorrer no cilindro OPC, como por exemplo, travamento mecânico do mesmo, onde se nota que a impressora trava com papel dentro do cilindro, quando isto ocorrer o cilindro também deve ser substituído.


Quando a impressora detecta o fim da vida útil do cilindro, irá aparecer a seguinte mensagem no painel: SUBST. DRUM ou CHG. DRUM, dependendo da língua em que a impressora estiver configurada, se o for confirmado que o cilindro de impressão está funcionando corretamente, basta efetuar o zeramento do contador do cilindro, este é feito pelo painel da seguinte forma:

· Ligar a impressora com a tecla “menu” pressionada o tempo necessário até que a mensagem “OPERACAO MNT” apareça no visor da impressora.
· Vá pressionando a tecla “menu” até que a mensagem “CNT DRUM REINICIO” apareça no visor.
· Pressionar a tecla “ENTER” do painel.
· A impressora reiniciará automaticamente e estará pronta.

Procedimentos para desligar corretamente um Nobreak

Você pode desligar o nobreak no botão ou interruptor apropriado ou retirando o plug de alimentação do nobreak, da tomada eletrica da rede.
Desta última maneira, qualquer nobreak irá interpretar que faltou energia!
Qual cuidado devemos então ter?
Quando falta energia, o nobreak passa à alimentar os equipamentos à ele ligados, com a carga contida em suas baterias. Esta carga é limitada, e não deve ser esgotada, sob risco de danificar as baterias.

Então, devemos "cortar" a alimentação vinda da beterias!

Como cortar a alimentação das baterias?

Grande parte dos nobreak´s tem uma chave, botão ou interruptor chamado de "inversor" , "inversão" ou ainda "baterias", o qual devemos primí-lo ou mudar de posição caso seja do tipo interruptor.

Como sei que o nobreak está totalmente desligado?

Nenhum alarme sonoro poderá estar disparado;
Nenhuma "luz" de sinalização deverá estar acessa.

Para religar o equipamento faça a operação inversa, conectar o nobreak a tomada da rede eletrica e religar a chave inversora.
Importante ressaltar, que há nobreak´s que fazem este último passo automaticamento, são maioria daqueles, cuja a chave inversora é um interruptor posional, bem parecidos com os interruptores usadas para iluminação (ligar e desligar lâmpadas).
Para quebrar células numa tabela no word

Em células cujo conteúdo se estende por mais de uma linha e que se situam próximas a uma quebra de página, por padrão o Word permite que a quebra de página ocorra no interior da célula, fazendo com que sua parte superior fique em uma página e a parte inferior na seguinte. Raramente é isso que o usuário deseja. Mas, caso você venha a se deparar com essa situação, não se preocupe que o problema é fácil de resolver. Basta selecionar a célula "quebrada", clicar no menu "Tabela", acionar a entrada "Propriedades da tabela", passar para a aba "Linha" e desmarcar o botão "Permitir quebra de linha entre páginas". Isso é o bastante para manter todas as células desta linha na mesma página.

Como desativar Firewall do XP
Qualquer firewall poderá ter incompatibilidades com aplicativos em algum momento, isso não é mérito só do Windows XP. O sistema operacional da Microsoft trata a segurança na internet com o firewall ICF. Esse software foi planejado para redes domésticas e de pequenas empresas, como é o seu caso, e funciona com LAN (rede local) ou conexões dial-up. Antes de desinstalar, é preciso averiguar algumas coisas importantes. Primeiro é necessário checar a idoneidade desse aplicativo recentemente instalado na empresa. Depois, é bom verificar se não foram instalados outros aplicativos, via internet, que possam estar comprometendo o bom funcionamento do firewall. Em terceiro lugar, é sempre bom estar com o sistema operacional atualizado, baixando o Service Pack da página do fabricante. Se ainda assim o problema continuar, avalie o quão importante é esse aplicativo novo, pois um firewall é uma ferramenta muito importante para a segurança da rede da sua empresa. Ele oferece proteção para os computadores diretamente conectados à internet e faz a verificação das portas e recursos (compartilhamentos de impressora e arquivo) de origem externa.
Se ainda assim você quiser desativar o firewall, basta seguir os seguintes passos:

- No menu iniciar, escolha Configurações e Painel de Controle.
- Dê um duplo clique em Conexões de Rede e Internet.
- Clique em Conexões de rede.
- Clique com o botão direito do mouse na conexão em que gostaria de desativar o ICF. Depois, clique em Propriedades.
- Na aba Avançada, clique na caixa para desmarcar a opção Proteger meu Computador ou Rede.

É sempre bom ressaltar que você precisará instalar um outro firewall para garantir a segurança.

Impressão do CADIN em máquinas com windows 2000 e impressora de rede Lexmark Optra M412
A aplicação do Banco Central - CADIN, utiliza o acesso VPN e é instalado em cada estação. O programa instalador encontra-se disponível para download na página do Banco Central, é o PASCS10. Ao acessá-lo, abre-se uma tela onde o usuário coloca sua identificação e senha; Nessa mesma tela, existe um campo com a informação: Conexão; ao clicar abaixo, há também uma opção para configuração da impressora. Ao clicar nessa opção, abrirá uma tela com os relatórios já gerados pelo usuário e uma opção para selecionar/configurar a impressora. Ao selecioná-la, é necessário utilizar um tamanho de fonte 8 (ou menos), quando na verdade a opção de Ajuda sugere a fonte courrier new 10 ou 13.
Nas máquinas com o win98, ao utilizar esse tipo de impressora, o relatório é automaticamente compactado, no win2000 isso não acontece, e como o cabeçalho do relatório é bastante extenso, ele só é impresso se o papel estiver como paisagem, o que pode não satisfazer ao nosso cliente.
Para entrar no sistema, deve-se voltar na tela de conexão.
Dentro do sistema, dependendo do tipo de informação que o usuário quer, o aplicativo mostra a tela e logo abaixo a opção IMPRIMIR. Em alguns casos, é necessário usar o F3, vai abrir uma tela pedindo o tipo de relatório, digita-se internet e ENTER. Essa função irá gerar relatórios no PASCS10 e a impressão dos mesmos deverá ser feita da forma acima explicado.

- Microsoft Office XP Service Pack 2
- Microsoft Project 2000 Standard Edition
- Microsoft Excel 2002 Standard Edition
- Microsoft Outlook 2002 Standard Edition
- Microsoft PowerPoint 2002 Standard Edition
- Microsoft Publisher 2002 Standard Edition

- Na caixa de diálogo Configuração do Microsoft Office XP, clique em Reparar Office e em Avançar.
- Clique em Reinstalar Office e em Instalar.
- Na caixa de diálogo Instalar Office, clique em OK.
- Caso solicitado que reinicie o computador para concluir a instalação, clique em Sim.
