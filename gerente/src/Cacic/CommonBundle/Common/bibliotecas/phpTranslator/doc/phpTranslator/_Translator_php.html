Translator
Classe para realizar a traducao de um texto com base em arquivos pre-definidos


Page Details:

Arquivo que contem a classe de traducao e outros para a traducao de aplicacoes

As aplicacoes baseadas nesse arquivo/classe deverao adaptar-se ao modo como este monta os arquivos de idiomas e seus requisitos de codificacao de idioma.

PHP versions 4 and 5
